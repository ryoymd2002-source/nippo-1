# 📋 現場日報（Construction Voice Nippo）

**スマホ一台で建設現場の日報作成〜管理を完結する**次世代型日報システム。

---

## 🎯 コンセプト

従来の日報作成は「現場で写真を撮る → 事務所に戻って印刷 → 糊で貼る → 手書きで説明を書く」という手間のかかる作業でした。

このシステムは **スマートフォン一台** で：
- 📸 写真を撮ってそのまま添付
- 🎤 音声入力で作業内容を記録
- 🤖 AIが自動で構造化
- ☁ クラウド保存でいつでもどこでも確認
- 📊 ダッシュボードで傾向分析

---

## ✨ 機能一覧

### 📝 日報作成（Create タブ）

| 機能 | 説明 |
|------|------|
| **音声入力** | マイクボタンを押して話すだけで日報を自動作成。AI（Gemini）が音声を解析し、現場名・天気・作業内容・材料・備考を自動抽出 |
| **手動入力** | 各フィールドに直接テキスト入力も可能 |
| **写真添付** | スマホカメラで撮影した写真をその場で添付。複数枚対応 |
| **天気自動取得** | 現在の天気・気温を自動で埋め込み |
| **現場名サジェスト** | 過去に入力した現場名が候補として表示 |
| **PDF出力** | 作成した日報をA4 PDFとしてダウンロード。現場への配布・印刷に便利 |
| **テンプレート保存** | よく使う作業内容・材料の組み合わせをテンプレートとして保存。次回からワンタッチで呼出 |

### 📊 履歴管理（History タブ）

| 機能 | 説明 |
|------|------|
| **一覧表示** | 保存した日報を日付順に一覧表示 |
| **カレンダー表示** | カレンダー上で日報の有無を視覚的に確認 |
| **現場進捗タイムライン** | 特定の現場に絞り込み、時系列で進捗をひと目で把握 |
| **フィルタリング** | 日付範囲・作成者・現場名で絞り込み |
| **全文検索** | 現場名・作成者・作業内容でフリーワード検索 |
| **詳細モーダル** | 日報をクリックで詳細表示。PDFダウンロードもここから |

### 🖼 フォトギャラリー（Gallery タブ）

| 機能 | 説明 |
|------|------|
| **現場別表示** | 現場ごとに写真をグループ化して一覧 |
| **日付別表示** | 日付ごとに写真をグループ化して一覧 |
| **ライトビュー** | 写真をタップで拡大表示、関連する日報情報も同時に確認 |

### 📈 ダッシュボード（Dashboard タブ）

| グラフ | 説明 |
|-------|------|
| 🌤 **天気内訳** | ドーナツ円グラフで天気（晴・曇・雨・雪）の割合を表示 |
| 🏗 **現場別ランキング** | 横棒グラフで現場ごとの報告数 TOP5 を表示 |
| 📈 **日別推移** | 折れ線グラフで日ごとの報告数推移を表示 |

### 🛠 ユーティリティ

| 機能 | 説明 |
|------|------|
| **CSVエクスポート** | フィルタリングした日報をCSVでダウンロード |
| **JSONバックアップ** | 全データをJSON形式でエクスポート |
| **データ削除** | 不要な日報を個別に削除（写真もクラウドから自動削除） |

## 🏗 システムアーキテクチャ

```
ユーザー（スマートフォン・PC）
    │
    ├── Next.js 14（SPA）
    │     ├── DailyReportApp.tsx（メインコンポーネント）
    │     ├── lib/csv-export.ts
    │     └── lib/weather.ts
    │
    ├── Supabase
    │     ├── Database（日報データ）
    │     │   └── daily_reports テーブル
    │     └── Storage（写真ファイル）
    │         └── report-photos バケット
    │
    ├── Gemini API（Google AI）
    │     └── 音声 → 日報のAI構造化
    │
    └── Vercel（ホスティング）
          └── 自動デプロイ（GitHub連携）
```

### 使用技術

| 技術 | 用途 |
|------|------|
| **Next.js 14** | フレームワーク（SPA + API Routes） |
| **TypeScript** | 型安全な開発 |
| **Supabase** | データベース（PostgreSQL）+ ファイルストレージ |
| **Gemini API** | AIによる日報構造化 |
| **recharts** | ダッシュボードグラフ描画 |
| **jsPDF + html2canvas** | PDF出力 |
| **Tailwind CSS** | UIデザイン |
| **Vercel** | ホスティング・自動デプロイ |

## 🚀 セットアップ

### 必要条件
- Node.js 18+
- Supabase アカウント（無料枠でOK）
- Google Gemini API キー（無料枠あり）

### インストール

```bash
git clone <repository-url>
cd nippo-app
npm install
```

### 環境変数

`.env.local` を作成:

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key

# Google Gemini API
GEMINI_API_KEY=your-gemini-api-key
```

### 開発サーバー起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

## 💡 従来の日報との比較

| 項目 | 紙 + デジカメ | Excel + メール | このシステム |
|------|-------------|---------------|------------|
| 📸 写真添付 | 撮影→印刷→糊付け（**15分**） | 撮影→PC転送→貼付（**10分**） | 撮影→保存（**10秒**） |
| 📝 記入 | 手書き（誤字・読めない） | キーボード入力 | 音声入力＋AI自動構造化 |
| 🔍 検索 | ファイルをめくる | フォルダを開く | ワンクリック即表示 |
| 👥 共有 | コピーして手渡し | メール添付 | URLを送信 |
| 💰 コスト | 紙代・インク代・印刷代 | ライセンス料 | **0円**（Supabase無料枠内） |
| 📱 携帯性 | デジカメ＋書類が必要 | PCが必要 | **スマホ一台で完結** |
